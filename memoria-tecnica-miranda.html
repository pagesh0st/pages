Renderizador 3D para Proyectos Académicos y Personales

Una herramienta multiplataforma desarrollada en Java y OpenGL para crear, modificar y renderizar escenas 3D con fines educativos, profesionales y de ocio.

Java OpenGL Dear ImGui GLSL Shaders

Características Principales

Multiplataforma

Desarrollado en Java, lo que permite ejecutar la aplicación en Windows, Linux y macOS sin modificaciones.

Interfaz Intuitiva

Utiliza Dear ImGui para una interfaz de usuario moderna y fácil de usar, accesible para todos los niveles de experiencia.

Efectos Gráficos Avanzados

Implementa shaders GLSL para aplicar efectos visuales avanzados a modelos y escenarios 3D.

Propósito Educativo

Diseñado específicamente para facilitar a estudiantes y profesionales la creación de recursos visuales para proyectos académicos y personales.

Descripción General del Proyecto

Propósito y Objetivos

Este proyecto consiste en un programa informático en el cual el usuario podrá realizar diversos efectos sobre modelos o escenarios 3D, con el propósito final de generar una imagen a partir de la escena en la que ha trabajado.

El objetivo principal es facilitar a estudiantes y profesionales una herramienta fácil de usar para que puedan generar imágenes y otros recursos para proyectos académicos, personales o por ocio.

Tecnologías Implementadas

Este programa está escrito en Java, un lenguaje multiplataforma, lo que lo vuelve relativamente fácil de exportar a varias plataformas de escritorio.

La herramienta utiliza la API de OpenGL para generar gráficos 2D/3D y rasterizarlos, mediante un abanico de herramientas, en las que podemos encontrar a los shaders, que son una parte fundamental dentro del proyecto.

Java OpenGL Dear ImGui GLSL Shaders

Público Objetivo

La aplicación está dirigida principalmente a:

  • Estudiantes de carreras relacionadas con informática, diseño gráfico o ingeniería que necesiten crear visualizaciones 3D para sus proyectos académicos.
  • Profesionales que requieran una herramienta ligera para prototipado rápido de escenas 3D.
  • Entusiastas del 3D y programación que deseen experimentar con efectos gráficos sin necesidad de software complejo.

Flujo de Trabajo

El flujo de trabajo típico dentro de la aplicación sigue estos pasos:

  1. Carga del modelo 3D: Importar un modelo desde archivo .obj o utilizar uno de los modelos predefinidos.
  2. Configuración de la escena: Ajustar iluminación, cámara y fondos según las necesidades del proyecto.
  3. Aplicación de shaders: Seleccionar y configurar shaders para aplicar efectos visuales al modelo.
  4. Previsualización: Visualizar en tiempo real los cambios realizados a la escena.
  5. Renderizado final: Generar la imagen final con la resolución y calidad deseadas.
  6. Exportación: Guardar la imagen renderizada en formatos populares como PNG, JPEG o BMP.
